# Terminal Tools 终端工具集

## 📖 简介

作为一名终端工具爱好者和 Neovim 用户，我深知优秀的终端工具能够极大地提升开发效率和使用体验。在日常工作中，我发现了许多令人惊艳的终端工具，它们不仅功能强大，而且设计精良。

创建这个仓库的目的是：

- 🎯 **收录优秀的终端工具**：整理和分享我在使用过程中发现的各类优质终端工具
- 📚 **提供使用参考**：为其他终端爱好者提供工具选择的参考
- 🤝 **促进交流分享**：建立一个终端工具爱好者的交流平台
- 💡 **持续更新维护**：随着新工具的出现不断更新和完善列表

## 🤝 如何贡献

我非常欢迎各位终端工具爱好者一起来完善这个列表！如果你有想要推荐的优秀终端工具，可以通过以下方式贡献：

### 贡献方式

1. **Fork 本仓库**
2. **创建你的特性分支** (`git checkout -b feature/add-new-tool`)
3. **在 `terminal_tools.md` 中添加你推荐的工具**
4. **提交你的更改** (`git commit -m 'Add: 添加 xxx 工具'`)
5. **推送到分支** (`git push origin feature/add-new-tool`)
6. **创建 Pull Request**

### 贡献指南

在添加新工具时，请确保包含以下信息：

- **工具名称**：清晰准确的名称
- **简短描述**：一句话说明工具的主要功能
- **GitHub 链接**：工具的官方仓库地址（如果有）
- **安装方式**：简要说明如何安装
- **分类**：将工具归类到合适的类别中

### 示例格式

```markdown
### [工具名称](GitHub链接)

简短的功能描述。

**安装方式**：`brew install tool-name` 或其他安装命令
```
